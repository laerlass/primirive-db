Учебная реализация легковесной документоориентированной базы данных на Python. Проект демонстрирует использование современных инструментов разработки (Poetry, Ruff) и продвинутых концепций языка: декораторов, замыканий и работы с файловой системой.

Возможности
    ● Управление таблицами: создание, удаление и просмотр списка существующих таблиц.
    ● CRUD-операции: добавление (Insert), чтение (Select), обновление (Update) и удаление (Delete) записей.
    ● Типизация: поддержка типов str, int, bool при создании схемы.
    ● Продвинутые фичи:
        ○ Автоматическая генерация ID.
        ○ Кэширование запросов select через замыкания.
        ○ Логирование времени выполнения операций.
        ○ Безопасное удаление с подтверждением (confirm action).
        ○ Централизованная обработка системных ошибок через декораторы.

Технологии
    ● Язык: Python 3.10+
    ● Менеджер зависимостей: Poetry
    ● Формат данных: JSON (для метаданных и записей)
    ● Линтер: Ruff (стандарт PEP8)
    ● Библиотеки: prettytable для отрисовки таблиц в CLI, prompt для удобного ввода.

Установка и запуск
    Убедитесь, что у вас установлен Python 3.10+ и Poetry.
    1. Клонирование и установка зависимостей:
        make install
    2. Запуск базы данных:
        make run
    3. Проверка кода (линтер):
        make lint
